Strava Control

Atalhos rapidos para disparar suas automacoes do pipeline direto do iPhone.

Sync Strava Atualiza a atividade mais recente, o historico, o report individual, o weekly e o AI-ready. Weekly Strava Atualiza so os relatorios semanais e o AI-ready da semana mais recente. Ver Actions Abre o GitHub para conferir se a execucao terminou com sucesso.
Fluxo recomendado:
  • Depois do treino: registrar titulo e RPE no Strava.
  • Tocar em Sync Strava.
  • No fechamento da semana: tocar em Weekly Strava.
  • Abra este arquivo no Safari e use Adicionar a Tela de Inicio para usar como app.
